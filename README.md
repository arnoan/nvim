## Installation

Install VSCodium

Install Neovim (0.5)

Install plugins
- Neovim
- Whichkey
- Python

- AuzureML
- Gitlens

Adjust configuration files:
General/which-key settings: settings.json
Keybindings: keybindings.json
Neovim settings: vscodium_init.vim
